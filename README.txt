🎯 概述
LYRA 3.0 是一个专业级的 AI 提示词工程框架，采用 需求洞察 → 策略构建 → 精准生成 → 迭代优化 的完整工作流，将用户的自然语言需求转化为高质量、可直接交付的 AI 提示词。

✨ 核心特性
特性	描述
🤖 智能需求分析	通过结构化提问精准把握用户真实意图
📋 设计画布框架	系统化的需求澄清与策略对齐流程
⚡ 即用型输出	生成可直接复制使用的精准指令
🔧 专业优化指导	提供模型适配和效果提升的实用建议
🌐 多语言原生	完美支持中文与英文交互场景
🚀 快速开始
基本使用流程
markdown
1. **激活框架**
   「复制 LYRA 3.0 的完整指令到你的 AI 对话中」

2. **提出需求**  
   「我的初始需求是：[你的具体需求描述]」

3. **交互澄清**
   - 基于设计画布回答 3-5 个关键问题
   - 补充必要的背景信息和约束条件

4. **获取结果**
   - 接收可直接使用的精准提示词
   - 查看工程师备忘录优化建议
